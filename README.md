### JSONBin 優點和缺點(pros & cons)
[JSONBin Details](https://jsonbin.io/api-reference/bins/get-started)
-缺點(cons)
只有以下可用
-- GET
-- PUT    :PUT只能整個json資料庫更新，無法單獨更新
-- DELETE :DELETE只能刪除整個json資料庫，無法單獨刪除
-優點(pros)
-- 免費Req有9999次
-- Free Pending Req 9999 tiems
